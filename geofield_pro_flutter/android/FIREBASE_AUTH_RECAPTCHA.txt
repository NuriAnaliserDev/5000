CONFIGURATION_NOT_FOUND — eng muhim sabab (tekshirildi)
================================================================
`firebase auth:export` buyrug‘i ham xuddi shu CONFIGURATION_NOT_FOUND bilan
yiqiladi. Bu shuni anglatadiki, muammo faqat SHA yoki google-services.json
emas — Firebase loyihasida AUTHENTICATION XIZMATI hali to‘liq yoqilmagan
yoki Email/Password yoqilmagan.

0) BIRINCHI NAVBAT (majburiy — brauzer)
   a) Quyidagi havolani oching (yoki open_firebase_auth.ps1 ishga tushiring):
      https://console.firebase.google.com/project/geofield-pro-8529f/authentication
   b) Agar “Get started” yoki birinchi marta ochiladigan ekran chiqsa —
      “Get started” ni bosing (Auth servisini Firebase tomonda ishga tushiradi).
   c) “Sign-in method” → Email/Password → yoqing → Saqlang.
   d) (Tavsiya) Google provayderini ham yoqing — reCAPTCHA / OAuth uchun foydali.
   e) Project settings → Android ilova → google-services.json ni qayta yuklab,
      android/app/google-services.json ni almashtiring.

1) TEKSHIRUV (ixtiyoriy, lekin tavsiya)
   Loyiha ildizidan:
   powershell -ExecutionPolicy Bypass -File android/verify_firebase_auth.ps1
   Agar “Auth sozlamalari OK” chiqsa, keyingi qadam: flutter run.

2) Android: default_web_client_id (reCAPTCHA / ijtimoiy emas, lekin tavsiya)
   a) Konsol: https://console.cloud.google.com/apis/credentials?project=geofield-pro-8529f
   b) "OAuth 2.0 Client IDs" da "Web client (auto created by ...)" ushbu loyiha
      client ID sini nusxalang (turi Web application).
   c) android/firebase_secrets.properties.example faylidan nusxa oling
      → android/firebase_secrets.properties nomiga, defaultWebClientId=... ni
      to‘ldiring. (Loyiha: build.gradle bu qiymatni R.string.default_web_client_id
      qilib yig‘ishda ishlatadi; google-services.jsondagi bo‘sh oauth_client o‘zini
      almashtirmaydi, shuning uchun u ikkala hollarda ham yordam beradi.)

3) Qayta build
   flutter clean && flutter pub get && flutter run

App Check ogohlantirishi (“No AppCheckProvider”) — alohida; birinchi navbatda
Authentication (0-bo‘lim) hal qilinishi kerak.
