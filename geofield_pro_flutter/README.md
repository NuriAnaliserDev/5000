# GeoField Pro Enterprise: Aurum Edition

**GeoField Pro** — bu "Aurum" tog'-kon kompaniyasi uchun ishlab chiqilgan professional geologik dala xaritalash, ma'lumotlarni yig'ish va tahlil qilish tizimi. Tizim ofis va dala muhandislarini yagona ekotizimda birlashtiradi.

## 🚀 Asosiy Imkoniyatlar

### 1. 🧊 Advanced 3D Visualization
- **3D Structural Viewer:** Strike va Dip o'lchovlarini yer ostidagi 3D tekisliklar shaklida ko'rish.
- **Topographic Integration:** Balandlik (altitude) ma'lumotlari asosida nuqtalarning o'zaro joylashuvini tahlil qilish.
- **Web & Mobile Sync:** Daladagi 3D model bir vaqtning o'zida ofisdagi Dashboardda ham aks etadi.

### 2. 🗺️ Professional GIS & Linework
- **KML/DXF Support:** GIS qatlamlarini import qilish va xarita ustida tahrirlash (Vertex editing).
- **Coordinate Conversion:** WGS-84 va UTM (Zone 42N/43N) tizimlari o'rtasida real-vaqtda konvertatsiya.
- **Cross-Sections:** Istalgan nuqtada relief va qatlamlarni kesib ko'rish (2D slices).

### 3. 🧠 Smart AI Reporting
- **OCR Analysis:** Qog'ozdagi qo'lyozma hisobotlarni (Spotter/Drilling log) kamera orqali tanish va raqamlashtirish.
- **Automated Mapping:** AI orqali Excel (Master Report) fayllarini avtomatik to'ldirish.
- **Verification Hub:** Web Dashboardda ma'lumotlarni tasdiqlash va tahrirlash markazi.

### 4. 🛰️ Safe Field Operations
- **Background Tracking:** 12 soatlik dala ishlarini avtonom kuzatib borish.
- **Geofencing:** Xavfli yoki bazadan uzoq hududlarga chiqqanda ogohlantirish.
- **SOS Terminal:** Favqulodda vaziyatlarda ofis bilan tezkor aloqa (geotag bilan).

## 🛠️ Texnologik Stack
- **Frontend:** Flutter (Web & Mobile)
- **Database:** Firebase Cloud + Hive (Offline-first caching)
- **AI:** Google Vertex AI (Gemini Flash/Pro models)
- **Maps:** Flutter Map + MBTiles (Oflayn xarita qatlamlari)

## 📦 O'rnatish va Ishga tushirish
1. `flutter pub get` — dependencylarni yuklash.
2. `flutter run` — Mobil ilovani ishga tushirish.
3. `flutter run -d chrome` — Web Dashboardni ochish.
